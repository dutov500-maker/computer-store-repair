# Инструкция по активации обхода страниц через Яндекс Метрику

## Проблема
Яндекс Вебмастер показывает: "Не включён обход по счётчикам"

## Решение

### Шаг 1: Войдите в Яндекс Вебмастер
1. Откройте https://webmaster.yandex.ru/
2. Выберите ваш сайт (комплаб.рф)

### Шаг 2: Найдите раздел "Индексирование"
1. В левом меню найдите раздел **"Индексирование"**
2. Выберите **"Обход по счётчику Метрики"**

### Шаг 3: Включите обход
1. Нажмите кнопку **"Включить обход"**
2. Подтвердите действие

### Шаг 4: Настройте параметры (опционально)
1. **Периодичность обхода**: Рекомендуем "Раз в неделю"
2. **Время обхода**: Выберите удобное время (ночные часы для меньшей нагрузки)
3. **Лимиты**: По умолчанию Яндекс установит разумные лимиты

## Что это дает?

✅ Яндекс будет автоматически обходить все страницы, на которых установлен счётчик Метрики  
✅ Повышается вероятность индексации всех страниц сайта  
✅ Быстрее обнаруживаются новые страницы  
✅ Улучшается актуальность данных в поисковой выдаче

## Уже сделано в коде:

✅ Яндекс Метрика установлена (счётчик 104704712)  
✅ Включены параметры trackLinks и trackHash для отслеживания переходов  
✅ Добавлен параметр ut:"noindex" для исключения служебных страниц  
✅ Настроен правильный возврат кода 404 для несуществующих страниц  
✅ Обновлен sitemap.xml с актуальными датами  
✅ Создана статическая страница 404.html для корректной обработки ошибок

## Проверка после включения

Через 1-2 дня после включения обхода:
1. Зайдите в Яндекс Вебмастер → Индексирование → Страницы в поиске
2. Проверьте, что количество проиндексированных страниц увеличилось
3. В разделе "Обход по счётчику" должна появиться статистика обхода

## Дополнительные рекомендации

1. **Убедитесь, что счётчик работает**: Яндекс Метрика → Отчёты → должны быть посещения
2. **Проверьте robots.txt**: Должен разрешать доступ Яндексу (уже настроено)
3. **Актуализируйте sitemap**: При добавлении новых страниц обновляйте даты в sitemap.xml

## Важно!

После включения обхода по счётчику, робот Яндекса начнёт посещать ваши страницы в течение 1-7 дней.
Первые результаты в виде увеличения проиндексированных страниц вы увидите через 1-2 недели.
